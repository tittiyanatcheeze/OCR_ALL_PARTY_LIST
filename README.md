# OCR รายชื่อผู้สมัคร ส.ส. แบบบัญชีรายชื่อ (Party List) 2569

โปรเจกต์นี้ใช้เทคนิค OCR (Optical Character Recognition) ร่วมกับ **Google Gemini API (gemini-2.5-flash)** เพื่อดึงข้อมูลรายชื่อผู้สมัครสมาชิกสภาผู้แทนราษฎร (ส.ส.) แบบบัญชีรายชื่อ จากเอกสาร PDF ของคณะกรรมการการเลือกตั้ง (กกต.)

## ข้อมูลใน Repository นี้
1.  **Source File:** `สส 4 ทับ 24.pdf` - ไฟล์ต้นฉบับจากเว็บไซต์ กกต.
2.  **Output File:** `output/allptl.csv` และ `output/allptl.xlsx` - ไฟล์ข้อมูลที่ดึงออกมาได้ อยู่ในรูปแบบ CSV/Excel พร้อมนำไปใช้งาน

**หมายเหตุ:** เพื่อความปลอดภัยและความเป็นส่วนตัวของข้อมูล ทางผู้จัดทำได้ทำการ **ลบข้อมูลที่อยู่ (Address)** ออกจากไฟล์ Output เรียบร้อยแล้ว เหลือเพียงลำดับ ชื่อ และพรรคการเมือง

## การทำงานของ Code
- `OCR.ipynb`: Notebook หลักที่ใช้ในการประมวลผลภาพและส่ง API ไปยัง Gemini
- `full_pipe.py`: สคริปต์ Python สำหรับ pipeline การทำงาน (ทางเลือก)
- การทำงานแบ่งเป็น 3 ขั้นตอน:
    1.  แปลง PDF เป็นรูปภาพ
    2.  ส่งรูปภาพไปให้ Gemini API ช่วยอ่านและจัดรูปแบบเป็น JSON
    3.  รวบรวมข้อมูลลงไฟล์ CSV/Excel

## Requirements
- Python 3.x
- Google Gemini API Key (หรือ OpenRouter)
- Libraries: `pandas`, `requests`, `pdf2image`, etc.

## Setup
1. Clone repository
2. ติดตั้ง dependencies
3. สร้างไฟล์ `.env` หรือกำหนด Environment Variable `OPENROUTER_API_KEY`
4. รัน `OCR.ipynb` หรือ `full_pipe.py`
